.. _XRD:

X-Ray Diffraction (XRD)
=====================================

This example shows how to use **pymatgen** to quickly estimate a powder diffractogram. You can also access this feature from the `AMSinput GUI <../../../GUI/index.html>`__.

.. note::

    You can install pymatgen into the AMS Python environment using the `package manager <../../../Installation/Optional_Components.html>`__.

.. seealso::

    See also the `pymatgen documentation for XRDCalculator <https://pymatgen.org/pymatgen.analysis.diffraction.xrd.html>`__.


To follow along, either

* Download :download:`xrd.py` (run as ``$AMSBIN/amspython xrd.py``).
* Download :download:`xrd.ipynb` (see also: how to install `Jupyterlab <../../../Scripting/Python_Stack/Python_Stack.html#install-and-run-jupyter-lab-jupyter-notebooks>`__ in AMS)
